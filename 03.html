Agents

interacts with an environment

uses sensors to perceive

uses actuators to act

has control policy connecting the sensors and actuators

Rational Agent

For each possible percept sequence, a rational agent should select an action that is expected to maximize its performance measure, given the evidence provided by the percept sequence and whatever knowledge the agent has.

Environment

Environments are the problems to which rational agents are the solutions.

PEAS

Performance Measure, Environment, Actuators, Sensors

Properties of Task Environments

Episodic vs Sequential

Sequential - short-term actions have long-term consequences

Episodic - no need to think ahead; one-time-only actions

Properties of Task Environments

Static vs Dynamic

Change in the environment while the agent deliberates for an action

Semidynamic - environment does not change, but the performance measure of the agent

Properties of Task Environments

Discrete vs Continuous

Number of states in the environment

Properties of Task Environments

Known vs Unknown

Knowledge of the governing 'rules' of the environment
