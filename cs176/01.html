CS 176: Digital Image Processing

Course Description

Digital Image Fundamentals. Image transformations and filters. Restoration and reconstruction. Compression. Recognition.

PREREQUISITE: CS 130

Course objectives



At the end of the course, the student must be able to:

  • Explain digital image fundamentals
  • Identify and apply the correct set and sequence of image transformations and filters to enhance images
  • Understand and implement basic image compression algorithms
  • Apply digital image processing techniques to recognize objects in images
